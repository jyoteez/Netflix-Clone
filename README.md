# Netflix Clone (Frontend) - MERN Stack

This is the **Frontend** of my Netflix Clone project built using **MERN** stack.  
It provides the UI for user authentication, movie listing, movie details, watching movie trailers (YouTube / embedded player), and more â€” similar to Netflix UI.

---

## Tech Stack (Frontend)

| Technology | Purpose |
|-----------|----------|
| React JS  | UI Library |
| React Router DOM | Page Routing |
| Axios     | API Calls |
| Context API / Redux | State Management (use as per your project) |
| Tailwind CSS / Material UI (optional) | Styling |
| TMDB API (optional) | Movie Data |
| Firebase/Auth0/JWT | Authentication UI Integration |

---

## Features

- Sign Up / Login UI
- Browse Movies & TV Shows
- Search Movies
- Watch Trailer (YouTube Player)
- Responsive Netflix UI
- Dynamic content fetched from backend
- Real-time movie cards like original Netflix
